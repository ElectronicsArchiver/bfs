
# Installation

*You can also **[Build]** it yourself.*

<br>

## Packages

<br>

### ![Badge Alpine]

```sh
apk add bfs
```

<br>

### ![Badge Debian] ![Badge Ubuntu]

```sh
apt install bfs
```

<br>

### ![Badge NixOS]

```sh
nix-env -i bfs
```

<br>

### ![Badge Void]

```sh
xbps-install -S bfs
```

<br>

### ![Badge FreeBSD]

```sh
pkg install bfs
```

<br>

### ![Badge MacPorts]

```sh
port install bfs
```

<br>

### ![Badge Homebrew]

```sh
brew install tavianator/tap/bfs
```
